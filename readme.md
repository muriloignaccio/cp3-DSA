# Sistema ATM em C

Projeto desenvolvido para a disciplina de Data Structure and Algorithms da FIAP.

O sistema simula um caixa eletrônico (ATM) executado via terminal/console, permitindo realizar operações bancárias básicas utilizando a linguagem C.

## Funcionalidades

O sistema possui as seguintes funcionalidades:

- Consultar saldo
- Realizar saque
- Realizar depósito
- Consultar extrato
- Validação de saldo insuficiente
- Validação de valores inválidos
- Limite diário de saque
- Histórico das últimas transações
- Menu interativo via console

## Tecnologias Utilizadas

- Linguagem C
- GCC Compiler
- Terminal/Console
